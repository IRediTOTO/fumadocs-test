CollectUI helps teams use HeroUI Pro v3 in local projects, CI/CD, and AI coding workflows.

It is not an account-sharing or official key-sharing service. Instead, it uses a proxy-based workflow:

- During Pro package installation, the worker downloads the latest tarball and wires packages such as `@heroui-pro/react` and `heroui-native-pro` into your project.
- During MCP initialization and tool calls, the proxy forwards requests with the upstream token and returns the upstream response.
- For AI agents, local MCP and Skills provide component knowledge and reusable implementation guidance.
